# POP Resize 640

Webアプリ用のPOP画像を  
**横幅640px / JPG / 半角ファイル名** に一括変換する、シンプルなブラウザツールです。

## 概要

POP Resize 640 は、店内POP画像を  
Webアプリにアップロードしやすい形式へ変換することを目的としたツールです。

- 画像は **サーバーへ送信されません**
- すべての処理は **ブラウザ内（ローカル）** で完結します
- GitHub Pages などの静的ホスティングで動作します

## 主な仕様

- 出力形式：**JPG**
- 画質：**quality 0.85**
- サイズ：**横幅640px固定**
  - 元画像が640px未満の場合は拡大しません
- ファイル名：**完全半角英数字**
  - `pop_YYYYMMDD_001.jpg` 形式
- 複数画像を **ZIPで一括ダウンロード**

## 使い方

1. サイトを開く
2. POP画像をドラッグ＆ドロップ、またはファイル選択
3. 「変換してZIPダウンロード」をクリック
4. 変換済みZIPを取得し、Webアプリへアップロード

## 想定用途

- Webアプリ用POP画像の事前加工
- 重い画像によるアップロードエラー防止
- 日本語ファイル名による不具合回避

## 技術仕様

- HTML / JavaScript
- Canvas API による画像処理
- JSZip（CDN）によるZIP生成
- サーバー通信・ログ保存なし
