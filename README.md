# POP Resize 640

Webアプリ用のPOP画像を  
**横幅640px / JPG / 半角ファイル名** に変換する、ブラウザ完結型ツールです。

複数画像をキュー形式で管理し、  
**ファイル名を1件ずつ確認・編集したうえでZIP出力**できます。

---

## 概要

POP Resize 640 は、外部依頼などで作成されたPOP画像を  
Webアプリに適した形式へ効率的に変換することを目的としたツールです。

- 画像は **サーバーにアップロードされません**
- すべての処理は **ブラウザ内（ローカル）** で完結します
- GitHub Pages などの静的サイトで動作します

---

## 主な機能

- 画像の追加（複数回追加・ドラッグ＆ドロップ対応）
- 選択済み画像を **一覧（キュー）表示**
- 出力ファイル名を **1件ずつ編集可能**
  - 半角英数字・記号のみ入力可（自動制御）
  - 拡張子は `.jpg` 固定
- ファイル名の重複チェック
- まとめて **ZIP形式でダウンロード**

---

## 出力仕様

- 形式：**JPG**
- 画質：**quality 0.85**
- サイズ：**横幅640px固定**
  - 元画像が640px未満の場合は拡大しません
- ファイル名例：
pop_new_20251215_001.jpg

---

## 使い方

1. サイトを開く
2. 画像を追加（複数回追加可能）
3. 必要に応じてタグ・日付・ファイル名を編集
4. 「まとめてZIP作成」をクリック
5. 変換済みZIPファイルを取得

---

## 想定用途

- Webアプリ用POP画像の事前加工
- 画像サイズ・ファイル名制約によるアップロードエラー防止
- 画像加工作業の時間短縮・負担軽減

---

## 技術仕様

- HTML / JavaScript
- Canvas API による画像リサイズ
- JSZip（CDN）によるZIP生成
- サーバー通信・ログ保存なし
